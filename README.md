# Robot Hand

플렉스 센서와 서보모터를 이용하여 동작 인식과 앱 연동을 통한 로봇 손
> 손가락 구부림 정도에 따른 플렉스 센서의 값으로 서보모터를 제어하며,
>
> 블루투스를 통해 원격 제어를 할 수 있습니다.
